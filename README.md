# Web Programming Final Project

웹프로그래밍실습 (SWE3048_42) Final Project

## 프로젝트 소개
Next.js를 사용해서 만든 Notion 사이트<br>

## 기능 소개
**기본 기능**
+ Login / Register
+ Add New Note
+ Modify Note
+ Delete Note

**추가 기능**
+ Search
+ Profile Image
+ Comment
+ Favorite and Pin
+ Theme / Font Change
+ Music Embed

## 실행 방법
```
npm install
npm run dev
```
<br> prisma db 실행
```
npx prisma studio
```
